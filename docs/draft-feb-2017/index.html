Feb 24, 2017 - Draft Release of the Swedish eID Framework

Draft release to the latest official release of the Swedish eID Framework. The major changes made since the Nov 2016-draft release are:

  • The specifications have been transformed from MS Word format into Markdown. All changes between versions can be listed using Git history. Any suggestions to the specifications can now be made using Git forks and pull requests.
    The specifications will from now on be published on http://elegnamnden.github.io/technical-framework.

  • A new specification has been added to the Swedish eID Framework, "ELN-0612 - BankID Profile for the Swedish eID Framework".

  • A section about Swedish eID:s being used to log in to foreign services was added to "ELN-0600 - E-legitimationsnämndens Tekniska ramverk".

  • Newly defined status codes was added to ELN-0602 - Deployment Profile for the Swedish eID Framework.

  • The "ELN-0606 - Entity Categories for the Swedish eID Framework" specification now defines the http://id.elegnamnden.se/st/1.0/public-sector-sp and http://id.elegnamnden.se/st/1.0/private-sector-sp service type categories. They are intended to indicate whether a Service Provider is a public or private sector SP.

  • Some modifications have been made to prepare for Swedish Identity Providers wishing to expose their services within the eIDAS Framework.

  • A set of new attributes were defined for eIDAS and use by BankID Identity Providers.

If you have comments on the draft specifications, please let us know by commenting on the draft on "E-legitimationsnämndens Tekniska Forum" - https://forum.eidasweb.se.

Specifications:

  • ELN-0600 - E-legitimationsnämndens Tekniska ramverk - version 1.5 (draft) (in Swedish)
  • ELN-0602 - Deployment Profile for the Swedish eID Framework - version 1.4 (draft)
  • ELN-0603 - Registry for Identifiers - version 1.4 (draft)
  • ELN-0604 - Attribute Specification for the Swedish eID Framework - version 1.4 (draft)
  • ELN-0606 - Entity Categories for the Swedish eID Framework - version 1.5 (draft)
  • ELN-0607 - Implementation Profile for using DSS in Central Signing Services - version 1.1
  • ELN-0608 - Certificate Profile for Central Signing Services - version 1.0
  • ELN-0609 - DSS Extension for Federated Signing Services - version 1.1
  • ELN-0611 - eIDAS Constructed Attributes Specification for the Swedish eID Framework - version 1.0 (draft)
  • ELN-0612 - BankID Profile for the Swedish eID Framework - version 1.0 (draft)
